EA F1 Hub (Kenya & East Africa)

Setup

1. Prereqs: Node.js 18+ and npm
2. Install deps:

```
npm install
```

3. Start Tailwind watcher (rebuilds CSS on save):

```
npm run dev
```

4. Open `public/index.html` with a static server. Example (auto opens browser on Windows):

```
npm run preview
```

Build

```
npm run build
```

This outputs `public/assets/tailwind.css` minified.
